# 🏧 ATM Simulation in JavaScript

A simple ATM simulation built using JavaScript. The application allows users to log in with a PIN, check their account balance, deposit money, withdraw money, and view transaction history.

## Features

* PIN Authentication
* Check Balance
* Deposit Money
* Withdraw Money
* Transaction History
* Input Validation
* Menu-driven Program

## Technologies Used

* JavaScript (ES6)

## Concepts Covered

* Variables
* Functions
* Arrays
* Loops
* Conditional Statements
* Switch Case
* User Input
* Basic Program Logic

## How to Run

1. Save the code as `atm.js`.
2. Run it in a JavaScript environment that supports `prompt()` and `alert()` (such as a browser or online JavaScript playground).
