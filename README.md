# 🏧 ATM Simulation in JavaScript

A simple ATM simulation built using JavaScript. The application allows users to log in with a PIN, check their account balance, deposit money, withdraw money, and view transaction history.

## Features

* PIN Authentication
* Check Balance
* Deposit Money
* Withdraw Money
* Transaction History
* Input Validation
* Menu-driven Program

## Technologies Used

* HTML
* JavaScript

## Project Structure

```text
ATM-Simulation/
│── index.html
│── atm.js
│── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Follow the on-screen prompts to use the ATM system.
